# Contributing to Fab

Anyone is welcome to contribute to the "Fab" project, provided they conduct
themselves in good faith and with respect for all.


## Support Requests


If you need help with something, please make use of the "Discussions" tab on
the GitHub project page. Do not jump straight to raising an issue, it may be
that your problem is easily answered by the community and does not require
changes to the code.


## Raising Bug Reports and Feature Requests


Bug reports and feature requests may be raised using the "Issues" tab, but
before you do, please check there isn't a similar closed issue. Also check
the "Discussion" tab to see if someone has already come across what you have
found.


## Contributions


Contributions may be made by raising a pull request against the trunk branch
at <https://github.com/MetOffice/fab>. You will need a personal fork of the
repository to host your development branch.

If this is your first contribution, your branch will need to include the
addition of your details to the the following list. If you are not submitting
contributions on behalf of an organisation please use "n/a" for your
affiliation.
